## 🛠️ Getting Started

```bash
git clone https://github.com/nguyenduchuynet/AI-SaaS-Chatbot.git
cd AI-SaaS-Chatbot
npm install
cp .env.example .env
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
